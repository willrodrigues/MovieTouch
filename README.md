# Movie Touch

This project has been built to show all Upcoming movies with a search option.
It's using the architecture Reactive MVVM (MVVM with RxSwift)

## Getting Started

To run this project, you need to clone this repository and checkout branch development as below instructions:

### Prerequisites

Cloning repository:

```
git clone https://github.com/willrodrigues/MovieTouch.git
```

Installing required PODs:

```
pod install
```

### Opening Project

Open the XCode by using the file (MovieTouch.xcworkspace)

As soon as the XCode has been openned, select the simulator and run the project.

## Running the tests

As soon as the XCode has been openned, select a simulator and execute tests by pressing keys:
Command + U
